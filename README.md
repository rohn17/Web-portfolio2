# Personal Portfolio Website

This is a simple personal portfolio website created using HTML, CSS, and JavaScript.  
The purpose of this project is to showcase my basic web development skills and academic projects.

## Features

- Simple and responsive portfolio layout
- Navigation menu for different sections
- About section with my background
- Projects section showing my work
- Contact form for user messages
- Interactive JavaScript features

## JavaScript Functionality

1. Users can add new benefits related to my hobby using an input box.
2. Each added item can be removed using a delete button.
3. When the contact form is submitted, an alert message is shown.
4. The footer dynamically displays the current date and time.

## Technologies Used

- HTML
- CSS
- JavaScript

portfolio

## How to Use

1. Download or clone this repository.
2. Open the project folder on your computer.
3. Open the **index.html** file in any web browser.
4. You will see the portfolio website.

### Interactive Features

- In the **Home section**, you can add a new benefit of coding using the input box and "Add" button.
- You can also remove any added item using the **Delete button**.
- In the **Contact section**, fill the form and click submit to see the alert message.
- The **footer automatically shows the current date and time**.
